Available Operations in Time-Based Greeting Program

Display Current Time
Shows the current system time in HH:MM:SS format.
Example: 14:35:20

Display Current Hour
Extracts and displays only the current hour.
Example: 14

Display Current Minute
Extracts and displays only the current minute.
Example: 35

Display Current Second
Extracts and displays only the current second.
Example: 20

User Hour Input
Allows the user to enter a custom hour value for testing greetings.
Example: Enter hour: 9

Morning Greeting
Displays "Good Morning Sir!" when the hour is between 0 and 11.

Afternoon Greeting
Displays "Good Afternoon Sir!" when the hour is between 12 and 16.

Night Greeting
Displays "Good Night Sir!" when the hour is between 17 and 23.

Greeting Time Ranges
0 – 11 → Good Morning
12 – 16 → Good Afternoon
17 – 23 → Good Night
